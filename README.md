# EscenaLocal

Proyecto correspondiente a la materia Práctica Supervisada, Tecnicatura Universitaria en Programación UTN-FRC 2025

### Resumen

La promoción de eventos musicales locales y la gestión de entradas suele realizarse de manera
fragmentada mediante redes sociales, mensajería informal y plataformas externas que no están
orientadas específicamente a la escena independiente. Esto dificulta la visibilidad de artistas
emergentes, la organización de eventos por parte de productores y el acceso del público a
información centralizada y confiable.

El objetivo de **EscenaLocal** es ofrecer una plataforma web que centralice la publicación de eventos
musicales locales, la visibilización de artistas y productores, y la venta de entradas mediante
pasarela de pago integrada. La aplicación busca facilitar la interacción entre los distintos actores del
ecosistema musical local (artistas, productores y público), optimizando los procesos de difusión,
organización y monetización de eventos culturales.

Palabras clave: aplicación web, eventos musicales, venta de entradas, artistas locales, productores
culturales.
